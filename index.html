GVSU-CIS641-ONETEAMONEGOAL

A WEBSITE TO BUY,SELL AND DONATE REUSABLE PRODUCTS

PROJECT OVERVIEW

This project is all about a website which is used to buy,sell and donate products that are old and reusable

Using Salesforce as a tool, to create a customized user interface in order to display the products such as old clothes,old toys and old books in the website.Salesforce communities enables us to create public websites and applications that are directly integrated with the Salesforce organization.
